## 表格

| 表头1 | 表头2 | 表头3 |
| ----- | ----- | ----- |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 | 

语法要点：
1. 表头和数据行之间必须有分隔线
2. 分隔线至少需要三个连字符 ---
3. 两端的竖线 | 是可选的，但建议保留以提高可读性
4. 不需要严格对齐，但对齐后更美观
