# Jupyter Notebook 常用快捷键（macOS 版）

___

## 一、两种模式  
命令模式（按 Esc 进入）：边框为蓝色   
编辑模式（按 Enter 进入）：边框为绿色

___


## 二、最常用快捷键（必记）
### 单元格操作

| 快捷键 | 功能 |
|--------|------|
| `Shift + Enter` | 运行当前单元格并跳转到下一个 |
| `Control + Enter` | 运行当前单元格（不跳转） |
| `Option + Enter` | 运行当前单元格并在下方插入新单元格 |
| `A` | 在上方插入新单元格（命令模式下） |
| `B` | 在下方插入新单元格（命令模式下） |
| `D + D` | 删除当前单元格（按两次 D） |
| `Z` | 恢复删除的单元格 |
| `C` | 复制单元格 |
| `V` | 粘贴单元格 |
| `Shift + V` | 粘贴到上方 |

### 单元格类型切换

| 快捷键 | 功能 |
|--------|------|
| `Y` | 转为代码单元格 |
| `M` | 转为 Markdown 单元格 |
| `R` | 转为 Raw 单元格 |

### 运行与输出

| 快捷键 | 功能 |
|--------|------|
| `O` | 切换输出折叠/展开 |
| `L` | 显示/隐藏行号 |

___

## 三、魔法命令（通用，在代码单元格中）

| 魔法命令 | 功能 |
|---------|------|
| `%run script.py` | 运行外部 Python 脚本 |
| `%load script.py` | 加载外部脚本到当前单元格 |
| `%time func()` | 单次计时运行 |
| `%timeit func()` | 多次运行取平均时间 |
| `%who` | 列出所有变量 |
| `%reset` | 清除所有变量 |
| `%history` | 查看历史输入 |
| `%pwd` | 显示当前目录 |
| `%cd /path` | 切换目录 |
| `%ls` | 列出目录内容 |
| `%matplotlib inline` | 使图表直接显示在 notebook 中 |

___

## 四、macOS 完整快捷键速查表

### 命令模式快捷键（按 `Esc` 进入）

| 快捷键 | 功能 |
|--------|------|
| `↑/↓` | 选择上/下一个单元格 |
| `Enter` | 进入编辑模式 |
| `Shift + ↑/↓` | 多选单元格 |
| `Shift + M` | 合并选中单元格 |
| `Control + Shift + -` | 在光标处拆分单元格 |
| `1~6` | 将 Markdown 单元格设为 1~6 级标题 |
| `H` | 显示所有快捷键帮助 |
| `I + I` | 停止内核（按两次 I） |
| `0 + 0` | 重启内核（按两次 0） |
| `Space` | 向下滚动 |
| `Shift + Space` | 向上滚动 |
| `Cmd + Shift + P` | 打开命令面板（部分版本） |

### 编辑模式快捷键（按 `Enter` 进入）

| 快捷键 | 功能 |
|--------|------|
| `Tab` | 代码补全/缩进 |
| `Shift + Tab` | 显示函数文档提示 |
| `Cmd + ]` | 增加缩进 |
| `Cmd + [` | 减少缩进 |
| `Cmd + A` | 全选 |
| `Cmd + Z` | 撤销 |
| `Cmd + Y` 或 `Cmd + Shift + Z` | 重做 |
| `Cmd + /` | 注释/取消注释 |
| `Esc` | 退出编辑模式 |
| `Control + Z` | 缩进（部分场景） |

---

## 五、macOS 特有注意事项

| 通用版快捷键 | macOS 适配 |
|-------------|------------|
| `Ctrl + Enter` | `Control + Enter`（不变，Control 就是 Ctrl） |
| `Alt + Enter` | `Option + Enter` |
| `Ctrl + ]` 缩进 | `Cmd + ]` |
| `Ctrl + /` 注释 | `Cmd + /` |

> **提示**：macOS 上 `Control` 键就是 `Ctrl`，位于键盘左下角。大部分 `Ctrl` 组合键保持不变，只有与系统冲突的（如 `Cmd+C` 复制）需要留意——在 Jupyter 编辑模式下 `Cmd+C` 仍可复制文本。
